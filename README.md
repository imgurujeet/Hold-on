## FAQ

#### Q1: Does HoldOn work when the app is closed?
Yes. HoldOn runs as a **foreground service**, so it continues monitoring charger status and pickpocket activity even if the app is closed.

#### Q2: Can I stop the alarm without authentication?
No. The alarm can only be stopped via **fingerprint, Face Unlock, or device PIN/Pattern/Password** for maximum security. You can authenticate directly from the lock screen.

#### Q3: Will the app drain my battery?
No. HoldOn is lightweight and optimized to monitor charging state and motion detection efficiently with minimal battery usage.

#### Q4: Can I adjust pickpocket sensitivity?
Yes. You can choose between **Normal, Crowded, High Security**, or set a fully **Custom** sensitivity level.

#### Q5: Can I customize the alarm sound?
Yes. You can select from available alarm sounds in settings. More sounds will be added in future updates.
